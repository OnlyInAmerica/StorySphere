Moved Files:
------------
Android Gradle projects use a different directory structure than ADT
Eclipse projects. Here's how the projects were restructured:

* AndroidManifest.xml => rajawali/src/main/AndroidManifest.xml
* assets/ => rajawali/src/main/assets/
* lint.xml => rajawali/lint.xml
* res/ => rajawali/src/main/res/
* src/ => rajawali/src/main/java/
